Code Reviews & Written Quizzes

Quizzes:

Every week, there will be a quiz to be taken in 1001G HN, on the lecture notes, reading, submitted programs, and programming assignments.
  • As the semester progresses, quizzes will include review questions as well as short programming exercises based on the homework.
  • Quizzes are on paper (echoing the style of the final exam and the first pass of many programming job interviews).
  • For the quizzes, you may not use any notes, books, or any computer device (laptop, phone, calculator, smart watch, etc.).
  • There are no make-up quizzes. Instead, the 10 highest quiz scores are used when computing the final grade.

Code Reviews:

Every week, there will be an in-person code review on the recently submitted programs.
  • For each code review, you choose one of the listed programs to explain on the whiteboard.
  • Code reviews are on the whiteboard (echoing the style of many programming job interviews).
  • All code reviews must be taken by the deadline.
  • There are no make-up code reviews. Instead, the 10 highest code review scores are used when computing the final grade.

Labs

Each week, you are expected to work through the associated Online Lab. Target dates for completing these are included. There is no deadline for completing these, but the programming assignments, quizzes and coding reviews are based on the information in the lab.

Lab 0: Set Up & Python Basics.   Target date: Wednesday, 31 January.

Lab 1: Submitting Programs & Collection Data Types.   Target date: Wednesday, 7 February.
Learning Objective: to use a linter to submit programs in good style and to understand the built-in collection data types.
