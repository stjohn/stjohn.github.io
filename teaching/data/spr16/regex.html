Name Data: The Social Security Administration keeps track of birth names by year and state. Before we work directly with the data, try the Baby Name Voyager visualization tool: What is the most popular girls’ name now? In 1990? In 1960? When did the name `Eden’ break into the top 1000 names? What occurs more: names starting with `C’ or with `J’? Using Regular Expressions for Name Data: Pattern What names will these match? Ed[ei]n Erin.* Stephan[(ie)y] Isabel{1,2}e* Ta[a-z]{4,10} [JY]asmine* [A-Z].*elle [A-Z][a-z]*[aey] Which of these patterns are easy to find using the name visualization? Which are hard? Why?
