A metric on a set X is a function (called the distance function or simply distance), d: X x X → R+, where R+ is the set of non-negative real numbers (because distance can't be negative so we can't use R), and for all x, y, z in X, the following conditions are satisfied:
  1. d(x,y) >= 0
  2. d(x,y) = 0 if and only if x=y
  3. d(x,y) = d(y,x)
  4. d(x,z) <= d(x,y) + d(y,z)
